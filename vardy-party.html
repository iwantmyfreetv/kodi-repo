1. Grab the repository zip and save it somewhere
  2. Pop open your pi using a file-explorer and go to one of these addresses:
    • //kodi
    • //kodi.local
    • //kodi.home


    Use "libreelec" for both the username and password if asked
  3. Drag the file repository.iwantmyfreetv-1.0.1.zip to the Downloads folder in Kodi
  4. In Kodi go to:
    1. Add-ons
    2. In the top-left click the Box icon
    3. Install from zip file
    4. Home folder
    5. downloads
    6. click "repository.iwantmyfreetv-1.0.1.zip"
    7. Install from repository
    8. iwantmyfreetv Add-on Repository
    9. Video add-ons
    10. Vardy Party
    11. Install

Job Done!
