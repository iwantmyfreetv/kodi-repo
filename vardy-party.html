1. Grab the repository zip and save it somewhere
  2. Pop open your pi using a file-explorer and go to one of these addresses:
    • //kodi
    • //kodi.local
    • //kodi.home

  3. Copy the file repository.iwantmyfreetv-1.0.1.zip to the pi and put in the Downloads folder
  4. In Kodi go to:
    1. Add-ons
    2. In the top-left click the box logo
    3. Install from zip file
    4. Home folder
    5. downloads
    6. click "repository.iwantmyfreetv-1.0.1.zip"
    7. Install from repository
    8. iwantmyfreetv Add-on Repository
    9. Video add-ons
    10. Vardy Party
    11. Install

Job Done!
